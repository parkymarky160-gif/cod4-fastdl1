-- Reign of the Undead --

This is the release of RotU v2.00 beta. 
It is beta in a way that not yet all features are complete (especially classes), and there will be bugs. If you find any, please report them 
on coddev.net. This package is meant for hosting purposes. If you wish to play LAN / SP you can wait for a later release
which will have an installer and a create server menu. Or you can execute the startmod.bat, which will load the mod for you
and you'll be able to start any map compatible with the mod.


-- Instalation --

Copy the rotu2beta folder into your call of duty 4 mod folder located at "...activision\call of duty 4\mods\".
Use the startmod.bat or load the mod from the mod list to play.

-- Things to remember (important!!!) --

	- Only maps specifically made for the mod will work
	- The bots take up player slots
	- Changing the map by the "map" command DOES NOT work!!! --> use change_map instead.
	- Do not try to kick bots
	- Kill # command kills a player or a zombie
	- kill_zombies 0 will kill all zombies.
	
-- Mapping --

Soon the development tools for Rotu2 will be updated and it'll be possible to make existing maps work with rotu2. Which means you do
not need the map source.

Visit www.coddev.net for maps/news and updates.
